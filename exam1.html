Changes I have made on the project

Loader

I added a loader on the pages while the user waits for the API to load. Added some circles with matching colour for the site. This helps the user to understand that the data are loading and will be displayed when the data are ready.

Category

I added a category filter on the blog post site. This helps the user to show the different post sorted by categories.

Header Image

I have only kept my header image on the homepage. This is because when the user click on the different pages the user do not need to scroll through the header image every time.

Stylesheet Fonts

I had multiply stylesheet for the fonts from Google fonts. So to prevent the browser to load through them all I collected them together to help the browser load more easily.

Title

I did not have a unique title on the specific blogpost pages. I added a code in the JavaScript to fix this problem. Now the user can se the title of the blogpost at the top of the browser and every page has a unique title.

H1

At every page I had the same h1. So to get a unique h1 on every page I changes this by fixing the code in the html to get h1 on the right title to get a unique h1.
