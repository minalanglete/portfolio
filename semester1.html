Changes I have made on the project

Carousel

I added a carousel at the homepage to display the different exhibition. I had a purple section displaying the different exhibition before, but it was difficult to read. So I made a more clean and neat design that was more easy to understand and use for the user.

#ID

I added a ID on the links at the different exhibition images at the carousel. This will help the user to get on the right place on the page. Because the ID is doing a jump to the part of the page that are relevant for the image the user clicks on.

Aside

The aside was floating on the site. To keep it separate from the rest of the page and get the user to understand that it is a sidebar I added a background colour to keep it together so it is not floating.

WCAG

I did have some bad contrast with the white and the pink colour. So I change the #cb55d4 to #e15eeb to make a better contrast.
